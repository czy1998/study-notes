# 常用包及其作用

### yargs-parser

把命令行参数转换为 json 对象，方便访问

```ts
// 假如终端输入
node bin/index.js name age -v 0.0.1 -location=china -money==0 -to=go

// bin/index.js
const yParser = require('yargs-parser');
const args = yParser(process.argv.slice(2));
console.log(args);

// 输出
{
  _: [ 'name', 'age' ],
  v: '0.0.1',
  location: 'china',
  money: '=0',
  t: true,
  o: 'go'
}
```
