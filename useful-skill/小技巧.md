### 小技巧

### macbook

```sh
    cmd + shift + c # 呼出终端
    cmd + shift + .(大于号那个键) # 显示隐藏文件/文件夹
```

### chrome

```sh
    cmd + shift + a # 快速搜索(切换)tab，可以跨窗口切换
```

### vs code

假定需要批量修改标签名称，即可选中一个标签，按下 F2 即可进行批量修改

### markdown

可以将左侧文件（例如图片 png），选中再按住`shift`，即可将整个文件拖入，自动换为 markdown 内引入图片的形式，还可以进行切换，替换成文件的相对路径
