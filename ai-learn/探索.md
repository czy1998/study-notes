# 探索

## github copilot

### 使用体验

> 个人偏好使用 `vscode` 开发，所以更习惯使用其内部的 ai 工具

- `tab 补全` 感觉挺不错的，就是 free 的补全次数有限，容易用完，需要升级
- `model` 支持也挺多的，最新的基本都有

### 获取途径

vscode 安装插件进行使用

1. free 计划，tab 补全挺不错，就是每月免费补全次数太少
2. 加入组织，共享组织的 copilot 有风险，可能会导致封号，不推荐
   - 若无违规，意外被封禁，可查看这里参考流程[进行申诉](https://linux.do/t/topic/32481)
3. 自费升级为 pro

## claude code

### 使用体验

> 目前无实际使用，暂不评价

### 获取途径

使用一些平台提供的 api-key 导入本地 shell 配置文件中，进行使用，此处以 zsh 为例

```ts
// .zshrc

export ANTHROPIC_AUTH_TOKEN=sk-xxx
export ANTHROPIC_BASE_URL=https://api.xxx.com/
```

1. [Any Router](https://anyrouter.top/)
   - 公益站点，每日登录可获取额度
   - [2026-1-9 11:02] 经测试，生成的 key 无法在 cc 中使用
2. [Code Router](https://api.code-relay.com/)
   - 目前只发现，注册送 $25，使用邀请码注册送 $5，其他需要充值购买
   - [2026-1-9 11:11] 经测试，生成的 key 可以在 cc 中使用
3. [XSTECH AI](https://xstech.one/)
   - 网页上直接使用，不能导出 api-key
4. 下面是一些`拼车`网站，还没用过
   - [Ai Code Next](https://www.aicodenext.com/guide.html)
   - [PinCC 官网](https://pincc.ai/join-carpool)
