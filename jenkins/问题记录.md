# 问题记录

## 1. 前端项目 jenkins 打包一直失败，本地打包就没有问题

**原因**

其中文件的引用路径不对，使用全小写，而实际路径是驼峰形式

mac 系统中打包没有问题

jenkins 上是 linux 系统，就会报错

**解决办法**

修正错误路径即可

## 2. leader 配置的 jenkins tool node 版本过高，老项流水线执行报错

**原因**

node 版本不一致，兼容性问题引起的

**解决办法**

1. jenkins 首页 -> Manage Jenkins -> 找到`全局工具配置` -> `nodeJS 安装` 里添加一个需要的 node 版本（可以指定是否使用镜像、全局依赖包）
2. 修改项目中的 Jenkinsfile tool 里使用的 node 即可
