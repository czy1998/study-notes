# Java

## 本地开发要求

1. jre
   安装 java 并不会安装 jdk
2. jdk
3. gradle

## idea 相关

1. 公众号 `mac管家` 可以下载
2. 古早项目需要自己下载 gradle 的话，需要在 idea 设置中，将 gradle 分发改成本地安装，并指定路径 (如：`/usr/local/gradle/gradle-9.6.1`)
3. 如果本地安装了 jdk，也可以指定 jdk
   `/usr/libexec/java_home -V` 可以查看本地安装的所有 jdk

## 插件

1. One Dark Theme
   累死 VS Code 的 One Dark 的主题
2. mybatisX
   可以帮你的 Mapper 文件添加链接，可以定位到对应的 xml
