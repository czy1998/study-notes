# 概念介绍

## BFF

【Backend For Frontend】: 为前端量身定制的后端服务

### 通常会做什么

- 聚合多个微服务数据
- 按前端需求裁剪/转换数据
- 处理鉴权、缓存、协议转换
- 屏蔽后端复杂性
- 错误码统一

### 技术栈选取

- Node.js
- NestJS / Express / Koa

## 网关

网关（Gateway）是计算机网络中的一种设备或服务器，用于连接不同网络或协议之间进行数据转发和处理。

### 一般做什么

- 路由（请求转发到哪个服务）
- 鉴权
- 限流
- 灰度
- 日志
- 监控
- 统一错误码

### 常见网关

- Nginx

## RPC

【Remote Procedure Call】: 远程过程调用，是一种计算机通信协议，它允许一个计算机程序通过网络调用另一个计算机程序中的子程序（也就是远程过程），并获取返回值。RPC 是一种分布式计算的重要基础，使得分布式计算应用更加方便和高效。

### 使用场景

- 后端服务之间调用
