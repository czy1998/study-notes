### linux 常用指令

```bash
cat > a.js # 新建文件，输入完内容 执行 ctrl+d 再回车 再 ctrl+d
cat a.js # 查看文件
cat a.js > b.js # 拷贝 a.js 的内容到 b.js 中，直接覆盖
cat a.js >> b.js # 拷贝 a.js 的内容到 b.js 的末尾，额外附加
cat -n a.js # 查看文件，显示行号

ls -l # 别名 ll，
ls -a # 输出目录和所有文件，包括隐藏文件

pwd # 显示当前所在目录

cd ~ # 切换到用户目录，直接输入 cd 也是一样效果
cd / # 切换到根目录

whoami # 当前登录用户名
```
