### ES Module

esm 是将 javascript 程序拆分成多个单独模块，并能按需导入的标准。和 webpack，babel 不同的是，esm 是 javascript 的标准功能，在浏览器端和 nodejs 中都已得到实现。使用 esm 的好处是浏览器可以最优化加载模块，比使用库更有效率。
