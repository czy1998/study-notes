```html
<div class="box">
```

```css
.box {
    max-height: 365px;
    width: 50px;
    overflow: auto;
    word-break: break-all;
}
.box::-webkit-scrollbar-thumb {
    background: rgba(255, 255, 255, 0.8);
}
```

上述代码即可将 box 的滚动条设置为灰白色
