# 应用

## vscode 中加入 mcp 服务器

这里以安装 `chrome-devtools-mcp` 为例子

1. 通过 vscode 拓展，mcp 市场中搜索安装

2. 项目根目录创建 `.vscode/mcp.json`，写入如下内容

   ```json
   {
     "servers": {
       "chrome-devtools": {
         "command": "npx",
         "args": ["-y", "chrome-devtools-mcp@latest"]
       }
     }
   }
   ```

### 可能会遇到的问题

1. 提示 node 版本过低

   - 使用 nvm 管理 node 版本
   - `nvm ls` 查看已有版本
   - `mvm install v22.21.1` 安装一个符合条件的 lts 版本
   - `nvm alias default v22.21.1` 将其设置为默认版本（避免出现使用 `nvm use v22.21.1` 切换版本后，运行还是出现版本不对的错误）
   - 然后彻底退出 vscode，重新打开 vscode，`cmd + shift + P` 选择 `MCP: List Severs` 列出 mcp 服务器，选择新加的，然后启动即可
